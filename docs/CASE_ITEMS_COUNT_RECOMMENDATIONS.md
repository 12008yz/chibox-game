# 📊 РЕКОМЕНДАЦИИ ПО КОЛИЧЕСТВУ ПРЕДМЕТОВ В КЕЙСАХ

## 🎯 ОПТИМАЛЬНЫЕ ДИАПАЗОНЫ (на основе психологии и практик индустрии)

### ✅ **ИДЕАЛЬНО: 15-30 предметов** (ЗОЛОТАЯ СЕРЕДИНА)

**Почему это оптимально:**
- ✅ Игроки могут мысленно оценить свои шансы (1 из 20 = 5%)
- ✅ Вероятности остаются понятными (от 0.5% до 15-20%)
- ✅ Легко контролировать RTP
- ✅ Достаточно разнообразия для интереса
- ✅ Визуально можно показать "редкость" (цвета, анимации)

**Примеры из индустрии:**
- CS:GO кейсы: **10-20 предметов**
- Dota 2 Treasure: **8-15 предметов**
- Valorant: **12-18 предметов**
- Mobile игры: **15-25 предметов**

---

### ⚠️ **ПРИЕМЛЕМО: 30-50 предметов** (для разнообразия)

**Когда использовать:**
- Дорогие кейсы (500₽+), где нужно больше вариантов
- Кейсы с широким ценовым диапазоном
- Специальные события (юбилейные кейсы)

**Ограничения:**
- Вероятности становятся меньше (0.1-5%)
- Нужно тщательнее балансировать веса
- Требуется более сложная визуализация

---

### ❌ **ИЗБЕГАТЬ: 50-200 предметов** (слишком много)

**Проблемы:**
- Вероятности микроскопические (0.0001-2%)
- Игроки не понимают свои шансы
- Размывается ценность предметов
- Сложно контролировать баланс
- Плохой UX (слишком много вариантов)

**Текущие проблемы в вашей системе:**
- Ежедневные кейсы: **171-199 предметов** ❌ (слишком много!)
- Премиум кейс: **126 предметов** ❌ (слишком много!)
- Стандартный кейс: **101 предмет** ❌ (слишком много!)

## 🧠 ПСИХОЛОГИЧЕСКИЕ ПРИНЦИПЫ

### 1. **Магическое число 7±2** (психология восприятия)
- Человек может одновременно удерживать в памяти **5-9 элементов**
- **15-30 предметов** = оптимально для восприятия категорий:
  - 5-7 дешевых (40-50%)
  - 5-7 средних (30-40%)
  - 3-5 дорогих (15-25%)
  - 1-3 джекпота (5-10%)

### 2. **Восприятие вероятностей**
- **<0.1%** = "невозможно" (игроки игнорируют)
- **0.1-1%** = "очень редко" (есть надежда)
- **1-5%** = "редко" (реалистичный шанс)
- **5-15%** = "иногда" (понятный шанс)
- **>15%** = "часто" (ожидаемо)

**Вывод:** Вероятности должны быть в диапазоне **0.5% - 15%** для психологического комфорта

### 3. **Эффект "близкого промаха"**
- Игрок видит предмет на 1 позицию выше/ниже выигрышного
- С **15-30 предметами** это работает лучше
- С **200 предметами** эффект теряется (слишком много вариантов)

## 💡 ПРАКТИЧЕСКИЕ РЕКОМЕНДАЦИИ

### ✅ **ЧТО ДЕЛАТЬ:**

1. **Уменьшить бесплатные кейсы до 20-40 предметов**
   - Мотивирует покупать платные
   - Упрощает балансировку
   - Улучшает UX

2. **Оптимизировать платные кейсы:**
   - Дешевые (17-101₽): **20-25 предметов**
   - Средние (250-601₽): **25-35 предметов**
   - Дорогие (998-2499₽): **30-40 предметов**
   - Премиум (5000-10000₽): **40-50 предметов**

3. **Использовать категоризацию:**
   - Показывать предметы группами (Common, Rare, Epic, Legendary)
   - Визуально выделять джекпоты
   - Анимация для редких предметов

### ❌ **ЧЕГО ИЗБЕГАТЬ:**

1. **Не делать кейсы с 100+ предметами**
   - Размывает ценность
   - Ухудшает восприятие вероятностей
   - Сложнее балансировать

2. **Не делать кейсы с <10 предметами**
   - Слишком предсказуемо
   - Быстро надоедает
   - Ограниченное разнообразие
